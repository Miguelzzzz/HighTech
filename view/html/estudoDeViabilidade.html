Estudo de Viabilidade

O que é o estudo de viabilidade?


O estudo de viabilidade é uma análise detalhada que avalia se um projeto é viável e sustentável,
levando em consideração custos, retorno financeiro, impactos ambientais e sociais.
Ele ajuda a minimizar riscos e embasar decisões, sendo fundamental para o sucesso de
qualquer empreendimento.

Viabilidade organizacional:

Trata do tema ou do modelo do negócio.

- O quê?

Trata-se de um website para ser utilizado como um ambiente
de teste para configuração de itens de acessibilidade digital;

- Para que?

Para estudos dos conceitos de desenvolvimento de projeto com recurso de acessibilidade;

- Como?

Projetando um website informativo que apresente textos sobre acessibilidade, explicação de
como o site foi criado e que ainda demonstre a aplicação de recursos de
acessibilidade com integração de bibliotecas externas/serviços externos.
